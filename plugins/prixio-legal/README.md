# Prixio Legal — Georgian IT Startup Legal Advisor Plugin

A comprehensive Claude Code plugin providing legal guidance for IT startups operating in Georgia (the country). Covers company formation, special tax regimes, intellectual property, data protection, employment law, and contract advisory.

## Skills

| Skill | Description |
|-------|-------------|
| `it-startup-law` | Special IT regimes (VZP, IC, GITA Startup, FIZ), tax comparisons, status application guides |
| `company-formation` | LLC/IE registration, House of Justice process, post-registration steps, foreign founders |
| `ip-protection` | Software copyright, patents, trademarks, trade secrets, work-for-hire, open source compliance |
| `data-protection` | Personal Data Protection Law (2023), GDPR-aligned compliance, privacy policies, breach handling |
| `employment-law` | Labor Code, employment contracts, remote work, foreign worker permits, non-compete, stock options |
| `contract-advisor` | Software development agreements, SaaS ToS, NDAs, investment agreements, e-commerce law |

## Agent

**`prixio-lawyer`** — Autonomous legal advisor that:
- Answers questions about Georgian law affecting IT companies
- Guides through company registration and status applications
- Advises on IP protection, data compliance, and employment
- Searches the web for legislative updates (matsne.gov.ge, gita.gov.ge)
- Responds in Georgian or English based on user's language

## Key Legal Sources

- [matsne.gov.ge](https://matsne.gov.ge) — Official legislative database
- [gita.gov.ge](https://gita.gov.ge) — Innovation & Technology Agency
- [sakpatenti.gov.ge](https://sakpatenti.gov.ge) — IP registration
- [pdps.ge](https://pdps.ge) — Data Protection Service
- [rs.ge](https://rs.ge) — Revenue Service

## Disclaimer

This plugin provides general legal information for educational purposes. It is NOT legal advice. Always verify with a licensed Georgian lawyer before making legal decisions.
