## About the Portal

infohub.rs.ge is the **Information and Methodology Hub** (საინფორმაციო და მეთოდოლოგიური ჰაბი) of the Georgian Revenue Service. It consolidates all tax and customs administration documents in one searchable space.

**Total documents**: 13,000+ across all categories
**Languages**: Georgian (ka), English (en)

## How to Use infohub.rs.ge

### Search
1. Go to https://infohub.rs.ge/ka
2. Use the search bar with keywords (Georgian recommended for best results)
3. Filter by document type, category, or search scope
4. Advanced search available for document numbers and consolidated changes

### Browse
1. Click "დათვალიერება" (Browse) tab
2. Select a document category
3. Browse or filter by subcategory/topic

### For Prixio Agent
When answering tax questions, reference specific guide numbers:
- For dividend questions → search "01xx" guides
- For VAT questions → search "11xx" guides
- For profit tax (Estonian model) → search "26xx" guides
- For penalty/reporting questions → search "23xx" guides
- For withholding tax → search "24xx" guides

Always provide the infohub link format:
`https://infohub.rs.ge/ka/workspace/document/[document-uuid]`
